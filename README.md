**Customer Churn Prediction & Retention Strategy**

This project predicts customer churn for an e-commerce platform and provides actionable retention strategies using machine learning. It analyzes customer purchase patterns, engagement levels, and behavioral features to identify customers at risk of leaving.

**Overview**

Predicts churn using ML models
Identifies key churn factors
Categorizes customers into Low, Medium, and High risk
Generates automated retention suggestions
Provides an interactive Streamlit dashboard for insights

**Dataset**

Includes:
Customer demographics
Purchase history & order frequency
Browsing behavior
Recency & engagement metrics
